# Travel Point
Premium travel agency landing page with multi-language support and interactive UI components.

## Tech Stack
-   **Core:** HTML5, CSS3 (Vanilla), JavaScript (ES6+)
-   **UI Framework:** [Bootstrap 5.3.2](https://getbootstrap.com/)
-   **Animations:** [Lottie-web](https://github.com/airbnb/lottie-web)
-   **Typography:** Unbounded (Google Fonts), Dot Matrix (Custom)

## Features
-   **Dynamic UI:** Interactive custom SVG cursor and animated page loader.
-   **LED Flight Board:** Custom-styled destination board with retro flicker animations.
-   **Multi-language (i18n):** Real-time language switching (EN/SK) using `translations.json`.
-   **Interactive Consultation:** Step-by-step form with Bootstrap accordion integration and progress tracking.
-   **Premium Aesthetics:** Dark-blue and peach color palette (#1F4E8C, #FFCFB9, #FFEC79) with AVIF image optimization.
-   **Component-based JS:** Modular script structure for footer injection and translation handling.

## Folder Structure
```text
travel-point/
├── index.html          # Main landing page
├── anketa.html         # Consultation form
├── thank-you.html      # Success page
├── stub.html           # "Coming Soon" page
├── style.css           # Global styles and responsive design
├── translations.json   # i18n data
├── assets/
│   ├── images/         # AVIF hero and background images
│   ├── icons/          # SVG icons (calendar, plane, etc.)
│   ├── lottie/         # JSON animation files
│   ├── logo/           # Brand assets and favicon
│   └── cursor/         # Custom SVG cursor asset
└── js/
    ├── cursor.js       # Custom cursor logic
    ├── footer.js       # Dynamic footer injection
    ├── i18n.js         # Translation engine
    └── loader.js       # Page loader functionality
```

# Travel Point
Преміальний лендінг для туристичної агенції з мультимовною підтримкою та інтерактивними елементами інтерфейсу.


## Технологічний стек
-   **Основа:** HTML5, CSS3 (Vanilla), JavaScript (ES6+)
-   **UI Фреймворк:** [Bootstrap 5.3.2](https://getbootstrap.com/)
-   **Анімації:** [Lottie-web](https://github.com/airbnb/lottie-web)
-   **Типографіка:** Unbounded (Google Fonts), Dot Matrix (Custom)

## Особливості
-   **Динамічне UI:** Інтерактивний кастомний SVG-курсор та анімований лоадер сторінок.
-   **LED Табло самольотів:** Стилізоване табло дестинацій з ретро-анімацією мерехтіння.
-   **Мультимовність (i18n):** Перемикання мов у реальному часі (EN/SK) через `translations.json`.
-   **Інтерактивна Консультація:** Покрокова форма з використанням Bootstrap accordion та відстеженням прогресу.
-   **Преміальна естетика:** Палітра у темно-синіх та персикових тонах (#1F4E8C, #FFCFB9, #FFEC79) з оптимізованими AVIF зображеннями.
-   **Модульний JS:** Структура скриптів для динамічної ін'єкції футера та обробки перекладів.

## Структура папок
```text
travel-point/
├── index.html          # Головна сторінка
├── anketa.html         # Форма консультації
├── thank-you.html      # Сторінка подяки

├── style.css           # Глобальні стилі та адаптивність

├── assets/

│   ├── icons/          # SVG іконки
│   ├── lottie/         # JSON файли анімацій
│   ├── logo/           # Логотипи та фавікон
│   └── cursor/         # Асет кастомного курсора
└── js/
    ├── cursor.js       # Логіка курсора
    ├── footer.js       # Динамічна ін'єкція футера
    ├── i18n.js         # Движок перекладів
    └── loader.js       # Функціонал лоадера сторінок
```
